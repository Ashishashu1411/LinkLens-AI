Privacy Policy for LinkLens AI

Last Updated: June 2026

1. Introduction

LinkLens AI is an AI-powered phishing URL detection and website security analysis platform available as both a web application and browser extension. The platform is designed to help users identify potentially malicious, phishing, and suspicious websites through machine learning and domain intelligence analysis.

This Privacy Policy explains how LinkLens AI collects, processes, stores, and protects information when users interact with the web application or browser extension.

2. Information We Collect

LinkLens AI only processes information necessary to perform website security analysis.

The platform may process:

  • URLs submitted by users for scanning
  • URLs of currently visited websites (only when browser extension scanning is enabled)
  • Scan timestamps
  • Threat classification results
  • Risk scores
  • Domain-related information obtained through WHOIS lookups
  • Application usage logs for debugging and performance monitoring

LinkLens AI does not collect:

  • Passwords, banking details, or personal messages
  • Email contents or credit card details
  • Personal documents or user credentials
  • Sensitive personal information

3. Purpose of Data Processing

Information is processed solely for:

  • Detecting phishing and malicious websites
  • Generating machine learning predictions
  • Performing domain intelligence analysis
  • Calculating website risk scores
  • Displaying security alerts and warnings
  • Improving system performance and detection accuracy
  • Maintaining scan history functionality

4. Browser Extension Permissions

The LinkLens AI browser extension may request permissions to:

  • Access the currently active tab URL
  • Analyze visited websites
  • Display security notifications
  • Store scan history locally
  • Communicate with the LinkLens AI backend service

These permissions are used exclusively for website security analysis and threat detection.

5. Data Storage

Scan history may be stored locally in the browser or application database to provide users with historical scan information.

Stored data may include: URL, scan result, risk score, and scan timestamp. No sensitive personal information is stored.

6. Third-Party Services

LinkLens AI may utilize third-party services including WHOIS lookup providers, domain intelligence services, cloud hosting, and database services solely to support cybersecurity analysis functionality.

7. Data Sharing

LinkLens AI does not sell, rent, trade, or share user information with advertisers, marketers, or unrelated third parties.

8. Security Measures

Reasonable technical and organizational safeguards are implemented to protect processed information from unauthorized access, data alteration, disclosure, or destruction. While reasonable security measures are maintained, no online system can guarantee absolute security.

9. User Rights

Users may clear locally stored scan history, disable automatic scanning, remove the browser extension at any time, stop using the web application, or request deletion of locally stored data.

10. Children's Privacy

LinkLens AI is not intended for use by children under the age of 13. The platform does not knowingly collect information from children.

11. Disclaimer

LinkLens AI provides risk assessments based on machine learning models, domain intelligence, and automated analysis techniques. Predictions are informational in nature and should not be considered absolute guarantees regarding website safety. Users should exercise their own judgment when visiting websites.

12. Changes to This Policy

This Privacy Policy may be updated periodically. Any modifications will be reflected on this page along with an updated revision date.
